Delete Hitcode

Flagging a hitcode is a signal to remove it at the time of the next LI system purge. It will no longer list by name in HAWB, but it can still be used until it is purged. If a hitcode already had an asterisk (*) to the left of it in the customer master maintenance, then it is flagged for deletion.
